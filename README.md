# github-actions-lab

PROG34104 — Activity 21: GitHub Actions CI/CD basics.

This repo contains a sample workflow that runs on pushes and pull requests to `main`.

## After cloning

1. Create a repository named `github-actions-lab` on GitHub (with or without a README).
2. Add this folder as a remote and push:

   ```bash
   git remote add origin https://github.com/<your-username>/github-actions-lab.git
   git branch -M main
   git push -u origin main
   ```

3. Open the **Actions** tab on GitHub to see workflow runs, logs, and step results.
